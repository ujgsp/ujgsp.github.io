Ticketing IT Helpdesk

January 22, 2025 #CodeIgniter

Projek: Ticketing IT Helpdesk

Deskripsi Singkat

Ticketing IT Helpdesk ini dapat membantu departement IT untuk mengelola permintaan tenaga IT dengan berdasarkan skala prioritas atau urgensi. Aplikasi ini terdapat 2 role admin, staff dan user.

Peran Saya

Web Developer

Teknologi yang Digunakan

  • Bahasa Pemrograman: PHP
  • Framework: CodeIgniter 3
  • Database: MySQL
  • Lainnya: UI Bootstrap, library spreadsheet dan datatable

Fitur dan Fungsi

  • Kelola User
  • Kelola Permintaan Ticket bantuan tenaga IT
  • Monitoring progress ticket
  • Approval berjenjang

Tautan atau Screenshot

  • Halaman Dashboard

  • Halaman Kelola Ticket

  • Halaman Kelola IT Solution
