Sistem Informasi Izin Proyek

March 3, 2025

Projek: Sistem Informasi Izin Proyek

Deskripsi Singkat

Aplikasi ini ditujukan untuk menunjuk vendor pekerjaan memulai suatu pekerjaan di lingkungan pekerjaan, karena secara prosedur setiap ada pekerjaan yang dilakukan oleh vendor harus mempunyai surat izin kerja dengan memenuhi standar keselamatan kerja.

Aplikasi memiliki beberapa role user dan role spv dan manager untuk approve.

Peran Saya

Web Developer

Teknologi yang Digunakan

  • Bahasa Pemrograman: PHP
  • Framework: CodeIgniter 3
  • Database: MySQL
  • Lainnya: UI Bootstrap, library Spreadsheet dan Fpdf

Fitur dan Fungsi

  • Kelola User
  • Pengajuan Surat Izin Kerja
  • Manajemen Approval Berjenjang
  • Generate laporan excel dan pdf

Tautan atau Screenshot

  • Form Pengajuan Izin Kerja

  • Form Detail Pengajuan Izin Kerja

  • Form Generate PDF Pengajuan Izin Kerja

Waktu Pengembangan
