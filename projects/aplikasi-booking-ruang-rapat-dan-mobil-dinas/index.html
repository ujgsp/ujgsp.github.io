Aplikasi Booking Ruang Rapat Dan Mobil Dinas

March 1, 2025 #CodeIgniter

Projek: Aplikasi Booking Ruang Rapat dan Mobil Dinas

Deskripsi Singkat

Aplikasi booking ruang rapat dan mobil dinas, monitoring ketersediaan ruang rapat dan mobil dinas dimana terdapat beberapa role dan approval kendaraan maupun ruangan.

Peran Saya

Web Developer

Teknologi yang Digunakan

  • Bahasa Pemrograman: PHP
  • Framework: CodeIgniter 3
  • Database: MySQL
  • Lainnya: UI Bootstrap, library spreadsheet dan datatable

Fitur dan Fungsi

  • Manajemen user
  • Manajemen pinjam ruangan rapat
  • Manajemen pinjam mobil dinas
  • Approval berjenjang
  • Laporan

Hasil dan Dampak

  • Efisiensi penggunaan Bahan Bakar
  • Efektivitas pengelolaan ruang rapat jika terdapat banyak ruangan
  • Kemudahan mengatur kendaraan dan ruangan
