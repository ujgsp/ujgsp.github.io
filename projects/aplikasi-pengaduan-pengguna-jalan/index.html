Aplikasi Pengaduan Pengguna Jalan

March 3, 2025 #CodeIgniter

Projek: Aplikasi Pengaduan Pengguna Jalan

Deskripsi Singkat

Untuk membangun brand image atau citra perusahaan serta dalam pemanfaatan teknologi dalam hal kepuasan pelanggan dibuatlah aplikasi berbasis web pengaduan pengguna jalan.

Sistem informasi ini memiliki 3 role: role user, role bujt dan role departement TMC sebagai approval atau verifikasi kelayakan pengaduan.

Peran Saya

Web Developer

Teknologi yang Digunakan

  • Bahasa Pemrograman: PHP
  • Framework: CodeIgniter 3
  • Database: MySQL
  • Lainnya: UI Bootstrap, notifikasi via bot telegram

Fitur dan Fungsi

  • Login User menggunakan OAuth
  • Manajemen User
  • Manajemen Pengaduan
  • Manajemen Gangguan
  • Informasi perkejaan yang sedang berlangsung

Tautan atau Screenshot

  • Login via OAuth

  • Halaman Dashboard

  • Kelola Pengaduan

  • Form input pengaduan

  • Form detail pengaduan

  • Profile User
